Como Usar o Sistema

Adicionar Nova Roupa:

  1. Clique em "Adicionar Nova Roupa"
  2. Preencha todos os campos obrigatórios
  3. Faça upload das fotos (mínimo 1, máximo 3)
  4. Selecione os tamanhos disponíveis
  5. Clique em "Salvar"

Gerenciar Roupas Existentes:

  1. Clique em "Ver Todas as Roupas"
  2. Use os botões de ação para cada roupa
  3. Edite, ative/desative ou exclua conforme necessário
  4. As alterações aparecem imediatamente na loja
